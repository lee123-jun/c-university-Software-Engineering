# 소프트웨어공학(Software Engineering)
- 소프트웨어 개발 프로세스, 개발 계획, 소프트웨어 분석/설계, 디자인 및 아키텍처, 소프트웨어 테스팅등의 소프트웨어 개발을 체계적으로 진행하기 위한 배경 지식을 학습한다. 소프트웨어공학의 기본이 되는 요구분석, 설계, 구현, 검증의 과정을 프로젝트를 SW의 분석 및 설계 방법을 실습한다.
